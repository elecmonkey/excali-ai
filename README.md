# Excalidraw AI Agent (excali-ai)

**AI-powered diagram editor that lets users create and modify Excalidraw diagrams through natural language. (Work in Progress)**

## Getting Started

You should have [Node.js](https://nodejs.org/) and [pnpm](https://pnpm.io/) installed.

1. Clone the repository & Install dependencies

```bash
git clone https://github.com/elecmonkey/excali-ai
cd excali-ai
pnpm install
```

2. Create a local environment file and configure your OpenAI-compatible API provider

```bash
cp .env.local.example .env.local
vim .env.local
```

3. Start the development server

```bash
pnpm dev
```
